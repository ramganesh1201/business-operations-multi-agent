# Project Summary

## Project Name

Business Operations Multi-Agent Copilot

## Problem

Business teams spend significant time processing emails, prioritizing tasks, and creating reports manually.

## Solution

A multi-agent AI system that automates:

* Email analysis
* Task extraction
* Task prioritization
* Report generation

## Technologies

* Python
* Gemini 2.5 Flash
* Google GenAI SDK
* MCP
* Streamlit
* GitHub

## Agents

### Coordinator Agent

Controls workflow execution.

### Email Agent

Extracts summaries, tasks, and deadlines.

### Task Agent

Prioritizes extracted tasks.

### Report Agent

Generates professional business reports.

## MCP Usage

BusinessFileServer provides:

* list_reports()
* read_report()

## Security

Role-based access controls restrict report operations.

## Outcome

Users can upload business emails and automatically receive prioritized tasks and professional reports.
