# Demo Script

## Introduction

This project is a Business Operations Multi-Agent Copilot built using Gemini AI, MCP, and Streamlit.

## Demo Flow

1. Open the Streamlit application.
2. Paste a business email.
3. Click "Process Email".
4. Coordinator Agent starts the workflow.
5. Email Agent extracts:

   * Summary
   * Action Items
   * Deadlines
6. Task Agent prioritizes tasks.
7. Report Agent generates a professional business report.
8. Report is saved to the reports folder.
9. Show generated report in the UI.

## Key Features

* Multi-Agent Architecture
* Gemini AI Integration
* Structured Agent Communication
* MCP Tool Layer
* Report Persistence
* Streamlit Interface
